# NarutoBrokenBond-Recompiled

### Naruto: The Broken Bond Recompilation Project

Welcome to the NarutoBrokenBond-Recompiled project!  
This project aims to bring *Naruto: The Broken Bond* to modern Windows PCs through Xbox 360 static recompilation research using XenonRecomp.

## Project Overview

*Naruto: The Broken Bond*, originally released for the Xbox 360 in 2008, remains one of the most requested Naruto games to preserve and restore for modern systems.

The goal of this project is to research and recompile the original Xbox 360 game code into a native executable capable of running on modern hardware while preserving the original gameplay experience.

## Project Status

The project is currently in early development and active research.
